# Avatar头像

用户头像的展览在

## 基本使用

可以开启行内样式，方便文件上传进度条

:::demo 可以使用 `gan-avatar`来加载一个头像，通过`src`指定头像图片，`name`指定头像名称
```html {2}
<template>
  <div>
  <gan-avatar src="https://dss1.bdstatic.com/70cFuXSh_Q1YnxGkpoWK1HF6hhy/it/u=3892521478,1695688217&fm=26&gp=0.jpg" name="Wangly"></gan-avatar>
  </div>
</template>
<script>
export default {
}
</script>
```
:::
